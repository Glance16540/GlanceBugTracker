Mustang is a premium Bootstrap Admin Template comes with Horizontal Navigation.

It is fully responsive, built with Bootstrap 3 Framework, HTML5, CSS3 and Sass. It has a huge collection of reusable UI components and integrated with latest jQuery plugins. It can be used for all type of web applications like custom admin panel, project management system, admin dashboard, Backend application or CRM.

This template includes around 30+ HTML pages and 500+ UI Components and widgets

Images in the live demo preview are demo purpose only, not included in the download folder. I have included placeholders in HTML files. Replace placeholder images as per your requirement.
